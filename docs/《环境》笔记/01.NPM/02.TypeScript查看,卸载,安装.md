# 如何查看 TypeScript 版本、卸载 TypeScript 和安装 TypeScript

### 1. 查看本机版本

```shell
npm view typescript version
```

### 2. 查看所有可以使用的版本

```shell
npm view typescript versions
```

### 3. 查看不同环境下的版本

```shell
npm info typescript
```

### 4. 查看本地是否安装

```shell
npm ls typescript
```

### 5. 全局查看是否安装，后面加个 -g

```shell
npm ls typescript -g
```

### 6. 卸载

```shell
npm uninstall typescript
```

### 7. 全局安装

```shell
npm install -g typescript
```
